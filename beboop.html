BE-BOOP BOOLEAN

Be-Boop Boolean is a game that was created as a simple, fun way to teach children about boolean login and promote Computer Science literacy from a young age. This game was created in Processing as my Programming Basics final project. The full game below has been translated into p5.js for web compatibility.

View the source code on my Github.

The inspiration for this game came from Minecraft's Hour of Code, Codeacademy, and other similar CS-related teaching tools. While the target audience is children, the goal of this project was to create something that would be informative, but still fun to play for all.
